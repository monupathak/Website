What I Did ?

Projects

Privacy Enhanced Movie Recommendation System

Developed a movie recommendation system using matrix factorization, which predicts what movies users might like based on past ratings. To protect user privacy, I added 3 different layers of differential privacy, ensuring that individual ratings can't be traced back to users.

AI for Advanced Data Analysis and Simulation

Created a framework to analyze and extract insights from large-scale simulation data using Data Farming and explainable AI techniques, with plans to integrate large language models for enhanced analysis.

Optimized Flight Delays using Heuristics

Developed a heuristic-based solution to minimize flight delays, generating Pareto optimal solutions using Gurobi Solver and further refining them with a threshold-based approach

LLM for Automated Commonsense Knowledge Graph Construction

Fine-tuned GPT-1 in COMET to automatically generate and enhance commonsense inferences, leveraging vast knowledge graphs,] achieving high-quality and novel commonsense generation.

Airline Ticket Sales Prediction Using Time Series Forecasting

Forecasted airline ticket sales using models like ARIMA, SARIMA, and Prophet, achieving superior accuracy with Facebook’s Prophet model by analyzing time series data for patterns and trends.

Vehicle Routing using Metaheuristics

Applied Genetic Algorithm and Genetic K-means metaheuristics to optimize vehicle routing, achieving optimal costs for different fleet sizes in a network with customer demand and depot coordinates.
